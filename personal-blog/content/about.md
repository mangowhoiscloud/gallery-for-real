# 안녕하세요, 반갑습니다!

저는 프론트엔드를 중심으로 풀스택 개발을 즐기는 개발자입니다.

## 기술 스택

주로 다음 기술들을 사용합니다:

- **언어**: TypeScript, JavaScript
- **프레임워크**: Next.js, React
- **스타일링**: Tailwind CSS
- **백엔드**: Node.js, PostgreSQL
- **인프라**: Vercel, Docker

## 이 블로그에 대해

이 블로그는 개발하면서 배운 것들, 삽질한 경험들, 그리고 가끔의 회고를 기록하는 공간입니다.

> 좋은 코드는 그 자체가 문서다 — 하지만 좋은 블로그도 필요하다.

주로 다루는 주제:

1. **개발** — Next.js, TypeScript, 아키텍처 패턴
2. **디자인** — CSS, 디자인 시스템, 사용자 경험
3. **회고** — 프로젝트 경험, 연간 회고
